## Ambient setup

1. Clone the repo  
   ```bash
   git clone https://github.com/lorenzoceglia/mp3-coverify.git
   cd mp3-coverify
   pnpm install
   ```

2. When adding or modifying a **feature**, make sure to also update the **README.md** to document the new or changed functionality.  
   Clear and up-to-date documentation helps others understand and use your contributions effectively.
